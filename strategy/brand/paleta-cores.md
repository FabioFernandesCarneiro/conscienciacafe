# Paleta de Cores - Consciência Café

Cores oficiais da marca extraídas do sistema de design do site.

---

## Cores Principais

### Preto (Primary)
```
Hex: #000000
RGB: 0, 0, 0
```
**Uso**: Textos principais, elementos de navegação, títulos, backgrounds premium.

### Laranja (Accent)
```
Hex: #FF4611
RGB: 255, 70, 17
```
**Uso**: Botões de ação (CTA), links, destaques, elementos interativos.

### Laranja Hover
```
Hex: #E53E0F
RGB: 229, 62, 15
```
**Uso**: Estado hover de botões e elementos interativos.

### Branco
```
Hex: #FFFFFF
RGB: 255, 255, 255
```
**Uso**: Fundos principais, textos sobre fundos escuros, espaço negativo.

---

## Cores Neutras

### Cinza Claro
```
Hex: #F5F5F5
RGB: 245, 245, 245
```
**Uso**: Fundos secundários, cards, seções alternadas.

### Cinza
```
Hex: #6B7280
RGB: 107, 114, 128
```
**Uso**: Textos secundários, legendas, metadados.

### Cinza Escuro
```
Hex: #374151
RGB: 55, 65, 81
```
**Uso**: Textos de suporte, ícones, bordas.

---

## Combinações Recomendadas

### Alta Contraste (Acessibilidade)
- Texto preto (#000000) sobre fundo branco (#FFFFFF)
- Texto branco (#FFFFFF) sobre fundo preto (#000000)

### CTAs e Destaques
- Botão laranja (#FF4611) com texto branco (#FFFFFF)
- Hover: (#E53E0F) com texto branco

### Conteúdo Secundário
- Fundo cinza claro (#F5F5F5)
- Texto cinza escuro (#374151)
- Metadados em cinza (#6B7280)

---

## Proporção de Uso

```
60% - Branco (fundos, espaços)
30% - Preto (textos, elementos)
10% - Laranja (destaques, CTAs)
```

---

## CSS Variables

```css
:root {
  --color-primary: #000000;
  --color-accent: #FF4611;
  --color-accent-hover: #E53E0F;
  --color-white: #FFFFFF;
  --color-gray-light: #F5F5F5;
  --color-gray: #6B7280;
  --color-gray-dark: #374151;
}
```
